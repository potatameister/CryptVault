About

CryptVault allows you to lock your text with a password so only you (or someone you trust) can read it. It runs locally on your device—no servers, no clouds.

How It Works

1. Key Generation

We take your password and a random "salt", then mix them 250,000 times using PBKDF2. This creates a unique mathematical key that is extremely hard to guess.

2. Encryption

We use the Web Crypto API, a native browser standard, to lock your message with AES-256. This is the same standard used by banks and governments.

Encryption Modes

AES-GCM (Recommended)

Galois/Counter Mode. It encrypts your data AND checks for tampering. If someone changes even one letter of the cipher, it will fail to decrypt.

AES-CBC

Cipher Block Chaining. A classic, robust standard. Good compatibility with older systems.

AES-CTR

Counter Mode. Very fast, turns block ciphers into stream ciphers. Great for large texts.

Triple Layer

Base64 encodes your data, encrypts it with AES-GCM, then Base64 encodes the result again. Overkill, but fun.

Simple (Deterministic)

Generates the encryption key directly from the password hash without a random salt. Same Password + Same Text = Same Result. (Use with caution).
